## 1. Project Overview

**Purpose:** A professional academic e-portfolio showcasing Ophelia's growth, competencies, and artifacts as an emerging student affairs professional, submitted in partial fulfillment of CSP 6050 Capstone requirements.

**Primary Goal:** Achieve the Exemplary tier on the PCA & Portfolio Grading Rubric (200 pts) by meeting all required elements with a creative, aesthetically engaging, and technically polished site.

**Technology:** Single-file HTML + CSS + JavaScript — no external hosting required, opens directly in any browser. Fully mobile responsive.

| Secondary / Headings | Warm dark brown | `#685D54` |

### Typography
- **Headings:** `Playfair Display` (serif, elegant, professional)
- **Body:** `Inter` or `Lato` (clean, highly readable sans-serif)
- **Accent/Quote:** `Playfair Display Italic`

### Design Language
- Clean whitespace — no clutter
- Subtle card shadows and hover effects
- Thin gold horizontal rules / dividers
- Smooth scroll and gentle fade-in animations
- Rounded card corners (8–12px)
- High contrast for accessibility

### Inspiration from Sample Sites
- *ccdvalentino*: Section-based single-page flow, professional headings, clear navigation
- *ashlbro*: Warm personal tone, bio-forward layout, artifacts clearly linked to competencies

## 6. Technical Specifications

| Requirement | Approach |
|---|---|
| Format | **Multi-page website** — separate HTML page per section (confirmed) |
| Platform | Pure HTML/CSS/JS — small folder of pages, no backend needed |
| Responsive | CSS Grid + Flexbox, media queries for mobile/tablet/desktop |
| Fonts | Google Fonts CDN (Playfair Display + Inter) |
| Icons | Font Awesome or Lucide (CDN) |
| Images | Placeholder `<div>` elements or `placehold.co` URLs — clearly labeled for replacement |
| Videos | Placeholder embed boxes (YouTube iframe slots) |
| Artifacts | Placeholder `href="#"` download links — to be updated with real file URLs |
| Navigation | Sticky top nav; mobile hamburger toggle |
| Accessibility | Alt text on all images, sufficient color contrast, keyboard navigable |
| File delivery | Saved to your portfolio folder; openable in Chrome/Safari/Edge |
